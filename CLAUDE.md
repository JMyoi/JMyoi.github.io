# Personal Site — CLAUDE.md

## Project
Personal portfolio/site built from scratch on GitHub Pages. This is a learning project — the primary goals are to relearn React and learn TypeScript.

See [PRD.md](PRD.md) for the full feature spec: sections, visual theme, animations, and open design questions.

## Tech Stack
- **Vite** — build tool and dev server
- **React** — UI framework
- **TypeScript** — language
- **TailwindCSS** — utility-first styling
- Additional styling libraries TBD

## Legacy Files
Old assignments have been moved to `oldAssignments/` and are **not part of the new project**. Leave them alone unless asked.

## Collaboration Rules
**This is the most important section.**

- **Do NOT write code** unless the user explicitly asks you to write it.
- When the user asks how to implement something, give a recommendation with best practices and explain the tradeoffs — then let the user write the code.
- When multiple approaches exist, pick one to recommend and explain why.
- The goal is learning. Guide and explain; don't do.

## Learning Goals
- Relearn React (components, hooks, state, props, routing, etc.)
- Learn TypeScript (types, interfaces, generics, utility types, etc.)
